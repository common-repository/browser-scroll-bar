1. Upload the plugin files to the `/wp-content/plugins/browser-scroll-bar` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Configure it (the plugin doesn\'t do anything until configured)
4. Verify that it works
